# ロードマップ & ビジョン

## エグゼクティブビジョン

TED Protocolは2026年までに10億ドル以上の資産を管理し、アクセシビリティ、透明性、パフォーマンスへのコミットメントを維持しながら、DeFiにおけるプレミア投資プラットフォームになることを目指しています。
